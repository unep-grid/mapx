# Map Data Attribution

This document is a general attribution guide for map products exported from MapX.

The exported ZIP also includes generated files based on the current printable map:

- `map_attributions.md`: a human-readable attribution checklist.
- `map_attributions.json`: structured attribution rows extracted from the map style and visible view metadata.
- `view_metadata_<id>.html`: detailed metadata for each visible view.

Printed map authors remain responsible for reviewing these files and including all required attributions, citations, licenses, and provider notices in the final product.

MapX terms of use are available at <https://docs.mapx.org/docs/legal-notices/terms-of-use.html>.
